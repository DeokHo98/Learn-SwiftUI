SwiftUI는 뷰 작업을 위한 많은 제스처를 제공하며 중요한 부분에 집중할수 있게 대부분의 힘든작업들을 대신 수행해준다.
이전에도 사용했던 onTapGesture 외에도 여러가지 방법이 있다.

struct ContentView: View {

    var body: some View {
        Text("tap tap")
            .onTapGesture(count: 2) {
                print("debug taptap")
            }
    }
}
onTapGesture에 매개변수를 전달하여 몇번눌러야 제스처를 처리할지를 결정할수도 있다.


struct ContentView: View {

    var body: some View {
        Text("tap tap")
            .onLongPressGesture(minimumDuration: 2) {
                print("debug taptap")
            }
    }
}
길게 터치를 최소 얼마부터 최대얼마까지 해야되는지도 정할수 있다.


struct ContentView: View {

    var body: some View {
        Text("tap tap")
            .onLongPressGesture(minimumDuration: 2) {
                print("debug taptap")
            } onPressingChanged: {
                print("debug \($0)")
            }
    }
}

또한 제스처 상태가 변경될때마다 트리거되는 클로저를 추가할수도 있다.





고급 제스처를 사용하려면 제스처 구조체중 하나와 함께 modifier를 사용해야한다.

제스처 구조체에는 DragGesture, LongPressGesture, MagnifyGesture, RotateGesture및 TapGesture가 있다.
이들 모두 일반적으로 onEnded와 onChanged modifier가 있으며
onEnded는 제스처가 완료됐을때 onChanged는 진행중일때 호출된다.


struct ContentView: View {
    @State private var currentAmount = 0.0
    @State private var finalAmount = 1.0

    var body: some View {
        Text("tap tap!")
            .scaleEffect(finalAmount + currentAmount)
            .gesture(
                MagnifyGesture()
                    .onChanged { value in
                        currentAmount = value.magnification - 1
                    }
                    .onEnded { value in
                        finalAmount += currentAmount
                        currentAmount = 0
                    }
            )
    }
}
핀치를 이용해 View를 확대 축소하는 예제



struct ContentView: View {
    @State private var currentAmount = Angle.zero
    @State private var finalAmount = Angle.zero

    var body: some View {
        Text("tap tap!")
            .rotationEffect(currentAmount + finalAmount)
            .gesture(
                RotateGesture()
                    .onChanged { value in
                        currentAmount = value.rotation
                    }
                    .onEnded { value in
                        finalAmount += currentAmount
                        currentAmount = .zero
                    }
            )
    }
}
두손가락으로 View를 회전시키는 예제


struct ContentView: View {
    var body: some View {
        VStack {
            Text("Hello, World!")
                .onTapGesture {
                    print("Text tapped")
                }
        }
        .onTapGesture {
            print("VStack tapped")
        }
    }
}
이런경우 어떤 TapGesture가 우선일까
항상 안쪽의 제스처가 우선순위다.
이른 반대로하는것도 가능한데
.highPriorityGesture 를 이용하면 된다.

struct ContentView: View {
    var body: some View {
        VStack {
            Text("Hello, World!")
                .onTapGesture {
                    print("Text tapped")
                }
        }
        .highPriorityGesture(
            TapGesture()
                .onEnded {
                    print("VStack tapped")
                }
        )
    }
}

또한 모두 실행하는것도 가능하다

                }
        }
        .simultaneousGesture(
            TapGesture()
                .onEnded {
                    print("VStack tapped")

}


마지막으로 가장 어려운 예제인데
어떤 제스처가 성공한경우 다른제스처또한 트리거 하게끔 하는것도 가능하다.
아래 예제에서는
롱 탭 제스처가 성공하면 그 후에 드래그할수 있게끔 구현하는 예제이다.

struct ContentView: View {
    // how far the circle has been dragged
    @State private var offset = CGSize.zero

    // whether it is currently being dragged or not
    @State private var isDragging = false

    var body: some View {
        // a drag gesture that updates offset and isDragging as it moves around
        let dragGesture = DragGesture()
            .onChanged { value in offset = value.translation }
            .onEnded { _ in
                withAnimation {
                    offset = .zero
                    isDragging = false
                }
            }

        // a long press gesture that enables isDragging
        let pressGesture = LongPressGesture()
            .onEnded { value in
                withAnimation {
                    isDragging = true
                }
            }

        // a combined gesture that forces the user to long press then drag
        let combined = pressGesture.sequenced(before: dragGesture)

        // a 64x64 circle that scales up when it's dragged, sets its offset to whatever we had back from the drag gesture, and uses our combined gesture
        Circle()
            .fill(.red)
            .frame(width: 64, height: 64)
            .scaleEffect(isDragging ? 1.5 : 1)
            .offset(offset)
            .gesture(combined)
    }
}
